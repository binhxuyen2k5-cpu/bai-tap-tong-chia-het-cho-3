# Bài tập 28 - Tính tổng số chia hết cho 3

Đây là ứng dụng web đơn giản dùng để tính tổng các số chia hết cho 3 trong khoảng từ số đầu đến số cuối do người dùng nhập vào.

## Cách sử dụng
1. Nhập số bắt đầu vào ô "Số đầu".
2. Nhập số kết thúc vào ô "Số cuối".
3. Nhấn nút **"Tính tổng"** để xem kết quả hiển thị ngay bên dưới.

## Công nghệ sử dụng
- HTML5
- CSS3
- JavaScript (ES6)
